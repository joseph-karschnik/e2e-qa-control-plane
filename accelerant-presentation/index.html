Command Quick Reference

Essential commands for the complete automation workflow

Test Generation

npm run jira:generate -- --work-item SF-520 Generate UI + API tests
npm run jira:generate -- --work-item SF-520 --type ui Generate UI tests only

Test Execution

npm run test:feature -- --tags "@SF-520" Run all tests for work item
npm run test:interactive Run with Playwright Inspector

Zephyr Integration

npm run zephyr:UploadTestCase -- --work-item SF-520 Upload test cases
npm run zephyr:UploadResult -- --work-item SF-520 Upload test results

Validation & Cleanup

npm run validate:steps -- --work-item SF-520 Validate step definitions
npm run cleanup:accounts Clean up test data
