Command Quick Reference

Essential commands for the complete automation workflow

Test Generation

npm run generate:tests -- --work-item PROJ-520 Generate UI + API tests
npm run generate:tests -- --work-item PROJ-520 --type ui Generate UI tests only
npm run ado:generate-tests -- --work-item PROJ-520 Generate SQL test cases from Azure DevOps work item
npm run sql:generate-comprehensive -- --work-item PROJ-520 Generate comprehensive SQL test cases with positive/negative scenarios

Test Execution

npm run test:feature -- --tags "@PROJ-520" Run all tests for work item
npm run test:interactive Run with Playwright Inspector

Test Case Management Integration

npm run tcm:UploadTestCase -- --work-item PROJ-520 Upload test cases (Zephyr / Xray / qTest)
npm run tcm:UploadResult -- --work-item PROJ-520 Upload test results

Validation & Cleanup

npm run validate:steps -- --work-item PROJ-520 Validate step definitions
npm run cleanup:accounts Clean up test data

Azure DevOps Integration

npm run azure-devops:fetch -- --work-item PROJ-520 Fetch work item from Azure DevOps
npm run azure-devops:test Test Azure DevOps connection

SQL Server Testing

npm run test:sql Test SQL Server connection
npm run sql:query -- --database "ODS" --query "SELECT * FROM Accounts" Execute SQL query for validation
